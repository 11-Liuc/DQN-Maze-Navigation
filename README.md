# 基于 DQN 的迷宫寻路智能体 (DQN-Maze-Navigation)

本项目是针对机器学习课程设计的实现，使用深度强化学习算法解决离散空间的寻路问题。

- [ ] DQN 算法实现
- [ ] 模型训练与调参

## 📂 目录说明
* `src/`: 核心源代码
* `docs/`: 实验报告与汇报材料
* `models/`: 训练好的模型权重 (`.pth`)
* `CONTRIBUTION.md`: 详细的小组分工表

1. 安装依赖：`pip install -r requirements.txt`
